#  正则表达式

## 1.re模块的基本使用

```python
import re
#方法一
result =re.match(正则表达式,字符串) #match能够匹配到以什么开头的字符串
result.groups()#返回所有匹配的字符
result.group(1)#返回对应分组匹配的字符
#方法二
p=re.compile(正则规则)
p.match("字符串")
```

##  2.正则规则

### 2.1匹配单个字符

| 字符 | 功能                          |
| :--: | :---------------------------- |
|  .   | 匹配任意1个字符（除了\n）     |
| [ ]  | 匹配[ ]中列举的字符           |
|  \d  | 匹配数字，即0-9               |
|  \D  | 匹配非数字，即不是数字        |
|  \w  | 匹配单词字符，即a-z、A-Z、0-9 |
|  \W  | 匹配非单词字符                |
|  \s  | 匹配空白，即 空格，tab键      |
|  \S  | 匹配非空白                    |

### 2.2 匹配多个字符

| 字符  | 功能                                                    |
| :---: | :------------------------------------------------------ |
|   *   | 匹配**前一个**字符出现0次或者无限次，即可有可无         |
|   +   | 匹配**前一个**字符出现1次或者无限次，即至少有1次        |
|   ?   | 匹配**前一个**字符出现1次或者0次，即要么有1次，要么没有 |
|  {m}  | 匹配**前一个**字符出现m次                               |
| {m,n} | 匹配**前一个**字符出现从m到n次                          |

### 2.3 匹配开头结尾

| 字符 | 功能           |
| :--: | :------------- |
|  ^   | 匹配字符串开头 |
|  $   | 匹配字符串结尾 |

###  2.4匹配分组

|   字符   | 功能                                                         |
| :------: | :----------------------------------------------------------- |
|    \|    | 匹配左右任意一个表达式                                       |
|   (ab)   | 将括号中字符作为一个分组                                     |
|   \num   | 引用分组num匹配到的字符串 **注意:**反向应用指的是前后内容完全相同,而能规则相同内容不同 |
| ?P<name> | 分组起别名**注意:**一般和()连用,且P要大写                    |
| ?P=name  | 引用别名为name分组匹配到的字符串                             |

### 3.2.贪婪和非贪婪

贪婪:匹配尽可能多的字符("*","?","+","{m,n}")*

python默认是贪婪的在"*","?","+","{m,n}"后面加上？使贪婪变成非贪婪

```python
>>> s="This is a number 234-235-22-423"
>>> r=re.match(".+(\d+-\d+-\d+-\d+)",s)
>>> r.group(1)
'4-235-22-423'
>>> r=re.match(".+?(\d+-\d+-\d+-\d+)",s)
>>> r.group(1)
'234-235-22-423'


In [1]: import re
In [2]: html = '<h1> hello world </h1>'    
In [3]: re.findall(r'<.*>', html)    # 贪婪模式默认匹配到所有内容
Out[3]: ['<h1> hello world </h1>']
In [4]: re.findall(r'<.*?>', html)   # 我们只想匹配两个标签的内容，可以加上问号来阻止贪婪模式
Out[4]: ['<h1>', '</h1>']

```

1. 假如你需要匹配文本中的字符反斜线，那么使用编程语言表示的正则表达式里将需要4个反斜杠`"\"`

   ```python
   2.正则表达式也使用反斜杠来转义字符
   ```

## 4.常见的正则用列

1.匹配出8到20位的密码可以是大小写英文字母、数字、下划线

```python 
re.match("[a-zA-Z0-9_]{8,20}","1ad12f23s34455ff66")
```

2.匹配163.com的邮箱地址

```python
re.match("[\w]{4,20}@163\.com$", email)
```

3.匹配出0-100之间的数字

```python
re.match("[1-9]?\d$|100,字符串)
```

4.匹配出163,126,qq邮箱

```python
re.match("\w{4,20}@(163|126|qq)\.com$,字符串)
>>匹配任意的邮箱
>>>r=re.match(r"^[0-9a-zA-Z]{0,19}@[0-9a-zA-Z]{1,13}\.[com,cn,net]{1,3}".text)
```

5.匹配出<html>hh</html>

```Python
re.match(r"<([a-zA-Z]*)>\w*</\1>", 字符串)
```

6.匹配出`<html><h1>www.itcast.cn</h1></html>`

```Python
re.match(r"<(?P<name1>\w*)><(?P<name2>\w*)>.*</(?P=name2)></?P=name1)>", label)
```

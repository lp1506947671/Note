#### 1.正则表达式

1.re模块

```python
import re
result =re.match(正则表达式,字符串) #match能够匹配到以什么开头的字符串
result.group()#获取匹配到的结果
```

2.匹配单个字符

| 字符 | 功能                          |
| :--: | :---------------------------- |
|  .   | 匹配任意1个字符（除了\n）     |
| [ ]  | 匹配[ ]中列举的字符           |
|  \d  | 匹配数字，即0-9               |
|  \D  | 匹配非数字，即不是数字        |
|  \w  | 匹配单词字符，即a-z、A-Z、0-9 |
|  \W  | 匹配非单词字符                |
|  \s  | 匹配空白，即 空格，tab键      |
|  \S  | 匹配非空白                    |

3.匹配多个字符

| 字符  | 功能                                                    |
| :---: | :------------------------------------------------------ |
|   *   | 匹配**前一个**字符出现0次或者无限次，即可有可无         |
|   +   | 匹配**前一个**字符出现1次或者无限次，即至少有1次        |
|   ?   | 匹配**前一个**字符出现1次或者0次，即要么有1次，要么没有 |
|  {m}  | 匹配**前一个**字符出现m次                               |
| {m,n} | 匹配**前一个**字符出现从m到n次                          |

4.匹配开头结尾

| 字符 | 功能           |
| :--: | :------------- |
|  ^   | 匹配字符串开头 |
|  $   | 匹配字符串结尾 |

5.匹配分组

|   字符   | 功能                                      |
| :------: | :---------------------------------------- |
|    \|    | 匹配左右任意一个表达式                    |
|   (ab)   | 将括号中字符作为一个分组                  |
|   \num   | 引用分组num匹配到的字符串                 |
| ?P<name> | 分组起别名**注意:**一般和()连用,且P要大写 |
| ?P=name  | 引用别名为name分组匹配到的字符串          |

6.re的高级用法  

| 1       | 2                                         | 3                                                          |
| ------- | ----------------------------------------- | ---------------------------------------------------------- |
| search  | 匹配出文章阅读的次数                      | re.search(r"\d+", "阅读次数为 9999")                       |
| findall | 统计出python、c、c++相应文章阅读的次数    | re.findall(r"\d+", "python = 9999, c = 7890, c++ = 12345") |
| sub     | re.sub(r"正则表达式", 新字符串, 老字符串) | re.sub(r"\d+", '998', "python = 997")                      |
| split   | re.split(r":\| ",字符串)                  |                                                            |

7.贪婪和非贪婪

贪婪:匹配尽可能多的字符
注意:python默认是贪婪的在"*","?","+","{m,n}"后面加上？，使贪婪变成非贪婪

```python
>>> s="This is a number 234-235-22-423"
>>> r=re.match(".+(\d+-\d+-\d+-\d+)",s)
>>> r.group(1)
'4-235-22-423'
>>> r=re.match(".+?(\d+-\d+-\d+-\d+)",s)
>>> r.group(1)
'234-235-22-423'
>>匹配任意的邮箱
>>>r=re.match(r"^[0-9a-zA-Z]{0,19}@[0-9a-zA-Z]{1,13}\.[com,cn,net]{1,3}".text)
```

#### 2.常见的正则匹配

1.匹配出8到20位的密码可以是大小写英文字母、数字、下划线

```python 
re.match("[a-zA-Z0-9_]{8,20}","1ad12f23s34455ff66")
```

2.匹配163.com的邮箱地址

```python
re.match("[\w]{4,20}@163\.com$", email)
```

3.匹配出0-100之间的数字

```python
re.match("[1-9]?\d$|100,字符串)
```

4.匹配出163,126,qq邮箱

```python
re.match("\w{4,20}@(163|126|qq)\.com$,字符串)
```

5.匹配出<html>hh</html>

```Python
re.match(r"<([a-zA-Z]*)>\w*</\1>", 字符串)
```

6.匹配出`<html><h1>www.itcast.cn</h1></html>`

```Python
re.match(r"<(?P<name1>\w*)><(?P<name2>\w*)>.*</(?P=name2)></?P=name1)>", label)
```

### 2.深拷贝和浅拷贝

|      | 浅拷贝                                                       | 深拷贝                                   |
| ---- | ------------------------------------------------------------ | ---------------------------------------- |
|      | b=copy.copy(a)                                               | b=copy.deepcopy(a)                       |
|      | 浅拷贝是对于一个对象的顶层拷贝                               | 深拷贝是对于一个对象所有层次的拷贝(递归) |
|      | copy.copy对于可变类型，会进行浅拷贝 <br>copy.copy对于不可变类型，不会拷贝，仅仅是指向 |                                          |



### 3.类中的双下划线,变量和函数

```python
# 双下划线变量
当父类中有双下划线变量时,继承父类的子类只能在创建对象进行初始化时才能赋值给父类的双下划线线变量
,不能通过子类的方法来改变双下划线变量
获取双下划线变量的方法:
class A:
    def __init__(self):
        self.__name = "text"

a = A()
print(a.__name)  # 报错
print(a._A__name)  # 输出text

# 单下划线变量
当父类中有单下划线变量时,即能在创建对象进行初始化时改变父类的双下划线线变量
,也能通过子类的方法来改变单下划线变量


# 注意:
继承:父类中双下滑线的变量,和方法,子类不继承,子类不能访问(单下划线变量和方法既能子类既能继承又能访问)
实例:类中双下滑线变量,和方法 ,实例不能访问(单下划线变量和方法既能实例能访问)
```
